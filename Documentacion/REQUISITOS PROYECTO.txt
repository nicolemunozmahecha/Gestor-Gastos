TALLER DE REQUISITOS PROYECTO

-----------------
REGISTRO DE GASTOS |
-----------------

REGISTRO:
Como usuario
Quiero registrar un gasto
Para controlar mis gastos personales

Criterios de adaptación:
- Se debe solicitar una categoría(alimentación, transporte, entretenimiento)
- Se debe solicitar el importe del gasto
- Se debe solicitar la fecha del gasto

----------
REGISTRO DE VARIOS USUARIOS CON CUENTA COMPARTIDA:
Como usuario
Quiero registrar gastos en una cuenta conjunta (no se pueden modificar)
Para repartir una serie de gastos

Criterios de adaptación:
- En tu aplicación debe haber una opción de añadir cuenta compartida
- Se introduce el nombre de los usuarios que la formen
- Esta cuenta debe ser mostrada como pestaña. Las cuentas que tiene un usuario se ven así.

REGISTRO DE GASTOS CON CUENTA COMPARTIDA:
Como usuario
Quiero registrar un gasto en una cuenta conjunta
Para repartir gastos

Criterios de adaptación:
- Se debe solicitar una categoría(alimentación, transporte, entretenimiento
- Se debe solicitar el importe del gasto
- Se debe solicitar la fecha del gasto

- Al introducir un gasto, el resto de personas de la cuenta deben pagar lo correspondiente al comprador. Cada persona deberá un saldo al comprador
- Se debe poder definir el porcentaje de gasto consumido por persona
 
-----------------------
CREACIÓN DE CATEGORIAS |
-----------------------
Como usuario
Quiero crear una nueva categoría  
Para registrar unos gastos específicos

Criterios de adaptación:
- Se debe elegir la opción crear categoría
- Se debe introducir el nombre de la nueva categoría

-------------

Para corregir errores

Criterios de adaptación:
- Se debe clicar en el gasto
- Se elige la opción modificar gasto
- Se modifica el campo
- Se guarda el cambio

-------------

Para corregir errores

Criterios de adaptación:
- Se debe clicar en el gasto
- Se clica en borrar gasto

----------------------------------------
FILTRAR GASTO POR MES, FECHA, CATEGORIAS|
----------------------------------------
FILTRO INDIVIDUAL:
Como usuario
Quiero filtrar un gasto
Para encontrarlo de manera más rápida

Criterios de adaptación:
- Se clica en un filtrar gasto
- Se elige el tipo de filtro (mes, fecha, categoría, DESPLEGABLE)
- Se escribe la correspondencia
- La opción debe estar bien escrita (p.ej: transporte, agosto, Agosto)
- La fecha debe ser válida

FILTRO CONJUNTO:
Como usuario
Quiero filtrar un gasto
Para encontrarlo de manera más rápida

Criterios de adaptación:
- Se clica en filtrar gasto
- Se eligen los filtros para filtrar
- Por campo: mes: mes válido; categoría: transporte

-------------------
ESTABLECER ALERTAS|
-------------------
Como usuario
Quiero establecer un limite de gastos
Para no gastar de más

Criterios de adaptación:
- Opción de alertas
- Se elige qué límite se quiere poner (semanal, mensual + categoría(opc))
- Si es por mes, configurar para ir haciendo el calculo del mes
- Se introduce el importe que no se quiere sobrepasar
- Si se excede el límite, sale mensaje de alerta
- Debe haber un historial de notificaciones

------------------
IMPORTE DE GASTOS |
------------------
Como usuario
Quiero importar un gasto
Para registrar más gastos de forma no individual

Criterios de adaptación:
- A la hora de registrar un dato debe estar la opción importar gastos
- Se elige un fichero de texto del ordenador
- Se lee el fichero linea por linea y se identifican los campos. Una vez identificados, por cada linea, se van introduciendo los gastos en la app.
- Se deben aceptar distintos formatos (orden de campos?)
